# 量子势垒隧穿模拟

高斯波包入射势垒的含时量子力学模拟。

## 在线运行（Binder）

点击下方链接即可在浏览器中运行，无需安装任何环境：

[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/<Zerwili>/<test>/HEAD)

## 本地运行

```bash
pip install -r requirements.txt
jupyter notebook
```

## Notebook 说明

| 文件 | 内容 |
|------|------|
| 01_theory_and_setup.ipynb | 理论推导与环境搭建 |
| 02_square_barrier.ipynb | 方势垒隧穿模拟 |
| 03_eckart_barrier.ipynb | Eckart势垒模拟 |
| 04_double_barrier.ipynb | 双势垒共振隧穿 |
| 05_comparison.ipynb | 三种势垒对比分析 |
| 06_dwell_time_hartman.ipynb | 驻留时间与Hartman效应 |
